1. Getting Started

Welcome to Chatbot Factory! This guide will help you get started with creating and managing your AI chatbots.

Account Registration Walkthrough

  1. Visit the main page and click on "Register"
  2. Fill in your email address and create a strong password
  3. Click "Register" to create your account
  4. Check your email for the OTP (One-Time Password)
  5. Your account will be in "pending" status until verified

üí° Tip

Use a valid email address that you check regularly. The OTP email usually arrives within a few minutes.

OTP Verification Process

  1. Check your email inbox for a message from Chatbot Factory
  2. Locate the 6-digit OTP code in the email
  3. Go to the OTP verification page
  4. Enter your email and the OTP code
  5. Click "Authenticate"
  6. Upon success, you'll be redirected to the login page

‚ö†Ô∏è Important

The OTP expires after a certain time. If your OTP doesn't work, you may need to register again or contact support.

Login and Logout Guide

Logging In:

  1. Click "Login" on the main page
  2. Enter your registered email and password
  3. Click "Login" to access your dashboard
  4. If credentials are correct, you'll be redirected to the dashboard

Logging Out:

  1. From the dashboard, locate the ‚ùå (logout) button in the top right
  2. Click the logout button
  3. You'll be logged out and can log in again later

First Time User Setup

After your first successful login:

  1. Review your account details on the dashboard
  2. Check your current plan and its limitations
  3. Familiarize yourself with the dashboard layout
  4. Consider upgrading your plan if you need more features
  5. Start by creating your first chatbot!

üéØ Quick Start

Begin with a simple conversational chatbot to understand the platform before creating more complex ones.

3. Creating Chatbots

Create powerful AI chatbots tailored to your specific needs with our easy-to-use interface.

Step-by-Step Guide to Create a New Chatbot

  1. Login to your dashboard
  2. Check your current chatbot count vs. plan limits
  3. Click "Create new Chatbot" button
  4. Choose a template (Conversational or Image Generation)
  5. Fill in the chatbot details form
  6. Click "Create" to generate your chatbot
  7. Wait for the creation process to complete
  8. Your new chatbot will appear in your dashboard

üí° Quick Creation

You can create a basic chatbot in under 2 minutes! Start simple and enhance later.

Choosing Chatbot Templates

Conversational Chatbot

Perfect for:

  • Customer service and support
  • FAQ assistants
  • General conversation practice
  • Information retrieval

Image Generation Chatbot

Perfect for:

  • Creative image generation
  • Design assistance
  • Visual content creation
  • Artistic projects

‚ö†Ô∏è Template Choice

Choose your template carefully! While you can edit settings later, the core functionality is determined by the template.

Customizing Chatbot Settings

When creating or editing a chatbot, you can customize:

  • Chatbot Name: A unique identifier for your bot
  • Chatbot Title: The display name users will see
  • Header Color: Customize the chat interface color
  • Background Color: Set the chat background color
  • Message Colors: Customize user and bot message colors
  • API Key: Your OpenAI API key for the bot
  • Model: Choose the AI model (GPT-3.5, GPT-4, etc.)
  • Credits Count: Set initial credits for the bot

Using the Chatbot Configuration Interface

The configuration interface allows you to:

  • Modify all visual aspects of your chatbot
  • Update API keys and AI models
  • Adjust credit allocations
  • Change colors and branding
  • Test your changes before saving

üé® Branding Tip

Use your company colors or create a unique theme for each chatbot to make them easily identifiable.

Publishing Chatbots

Once your chatbot is created:

  1. Go to your dashboard
  2. Find the chatbot in your list
  3. Click "Open" to access the chat interface
  4. Share the URL with users (each chatbot has a unique link)
  5. The chatbot is immediately available for use

Deleting Chatbots

To delete a chatbot:

  1. Go to your dashboard
  2. Locate the chatbot you want to delete
  3. Click "Edit" next to the chatbot
  4. Look for the delete option (usually at the bottom)
  5. Confirm the deletion

‚ö†Ô∏è Deletion Warning

Deleting a chatbot is permanent! All data, conversations, and configurations will be lost. Consider backing up important information first.

4. Persona Management

Personas allow you to create chatbots with distinct personalities and consistent behavior patterns.

‚ö†Ô∏è Personas Requirements

Personas are only available on Gold and Platinum plans. Upgrade your plan if you need persona functionality.

Understanding Personas

A persona is a character profile that defines:

  • Personality traits and characteristics
  • Communication style and tone
  • Specific knowledge areas or expertise
  • Consistent behavior patterns
  • Visual appearance and branding

Creating New Personas

  1. Ensure you're on a Gold or Platinum plan
  2. From your dashboard, click "Create new Persona"
  3. Fill in the persona details form with:
    • Persona name and description
    • Personality traits and background
    • Visual customization (colors, avatar)
    • API key and model selection
  4. Click "Create" to generate your persona
  5. Your persona will appear in the dashboard

Editing Existing Personas

  1. Go to your dashboard
  2. Find the persona in your list
  3. Click "Edit" next to the persona
  4. Modify the desired settings
  5. Click "Update" to save changes

Using Personas with Chatbots

Personas can be applied to chatbots to give them consistent personalities:

  1. Create or edit a chatbot
  2. In the configuration, select an available persona
  3. Save the chatbot settings
  4. The chatbot will now respond according to the persona's traits

üé≠ Persona Benefits

Personas are perfect for creating consistent brand voices, character-based chatbots, or specialized expert systems.

Persona Templates Overview

Available persona templates include:

  • Professional Assistant: Formal, helpful, business-oriented
  • Friendly Guide: Casual, encouraging, educational
  • Tech Expert: Knowledgeable, precise, technical
  • Creative Partner: Imaginative, collaborative, artistic
  • Customer Service: Patient, solution-focused, polite

5. Using Your Chatbots

Learn how to effectively use and manage your deployed chatbots.

Accessing Your Chatbot Dashboard

From your main dashboard:

  1. Scroll to the "Your Chatbots" section
  2. Find the chatbot you want to access
  3. Click "Open" next to the chatbot name
  4. The chat interface will open in a new tab

Interacting with Conversational Bots

Conversational bots respond to text input:

  • Type your message in the input field
  • Press Enter or click send
  • Wait for the AI to generate a response
  • Continue the conversation naturally
  • Use the copy feature to save important responses

üí¨ Conversation Tips

Be clear and specific in your questions. The AI performs better with well-formulated queries.

Using Image Generation Features

Image generation bots create visual content:

  • Describe the image you want in detail
  • Include style, colors, composition details
  • Wait for the AI to generate the image
  • Download or save the generated image
  • Refine your description for better results

Understanding Credit Usage

Credits are consumed based on activity:

  • Text messages: 1-5 credits per exchange
  • Image generation: 10-50 credits per image
  • Complex queries: 5-15 credits
  • Long conversations: Credits accumulate

‚ö†Ô∏è Credit Monitoring

Monitor your credit usage regularly. When credits reach zero, the chatbot will stop responding until upgraded.

Viewing Conversation History

Depending on your plan:

  • Context memory maintains recent conversation history
  • Disk memory (Gold/Platinum) stores conversations persistently
  • Use the copy feature to save important exchanges
  • Export conversations (Silver and above) for record keeping

Managing Resident Memory

Resident memory stores conversation context:

  • Larger memory allows for longer, more coherent conversations
  • Memory usage varies by plan (Free: basic, Gold/Platinum: extended)
  • Clear memory periodically to improve performance
  • Monitor memory usage in the chatbot settings

6. Credit System & Usage

Understanding how credits work is essential for managing your chatbots effectively.

How Credits Work

Credits are the currency that powers your chatbots:

  • Each plan includes a specific credit allocation
  • Credits are consumed when chatbots interact with AI services
  • Credits are allocated per chatbot, not per account
  • When credits run out, chatbots stop functioning
  • Credits do not automatically renew (except with plan upgrades)

Credit Allocation Per Plan

Plan Total Credits Per Chatbot Approx. Text Messages
Free 1,000 1,000 200-300
Silver 10,000 5,000 1,000-1,500
Gold 30,000 7,500 1,500-2,000
Platinum 60,000 10,000 2,000-3,000

Using Credits Effectively

  • Be concise: Shorter messages use fewer credits
  • Batch requests: Combine related questions
  • Use appropriate models: GPT-3.5 uses fewer credits than GPT-4
  • Monitor usage: Check credit levels regularly
  • Plan ahead: High-traffic periods may require more credits

Monitoring Credit Usage

Track your credit usage:

  1. Go to your dashboard
  2. View your chatbot list
  3. Check the "Chatbot Credits" field for each bot
  4. Monitor usage patterns over time
  5. Set up alerts if available on your plan

üìä Usage Patterns

Most users find that credit usage is highest during the first week as they test and optimize their chatbots.

Recharging or Upgrading

When credits run low:

  1. Option 1: Upgrade to a higher plan for more credits
  2. Option 2: Reduce usage on current chatbots
  3. Option 3: Delete and recreate chatbots (resets some credits)
  4. Option 4: Contact support for credit packages

‚ö†Ô∏è Credit Exhaustion

When a chatbot runs out of credits, it will stop responding. Always monitor credit levels to avoid service interruption.

7. Settings & Account Management

Manage your account settings and preferences to keep your profile secure and up-to-date.

Changing Password

To change your password:

  1. Login to your dashboard
  2. Look for account settings (future feature)
  3. Enter your current password
  4. Enter your new password twice
  5. Save changes

üîí Password Security

Use strong passwords with at least 8 characters, including uppercase, lowercase, numbers, and symbols.

Managing API Keys

API keys connect your chatbots to AI services:

  • Each chatbot can have its own API key
  • Use OpenAI API keys for GPT models
  • Keep API keys secure and private
  • Rotate keys regularly for security
  • Different keys can have different rate limits

Resetting Password (Forgot Password Flow)

If you forget your password:

  1. Contact support at the provided email
  2. Verify your identity with your registered email
  3. Support will guide you through the reset process
  4. Set a new strong password
  5. Login with your new password

‚ö†Ô∏è Account Recovery

Always use a valid, accessible email address. Account recovery depends on email verification.

Viewing Account Statistics

Your dashboard shows comprehensive statistics:

  • User ID: Your unique identifier
  • Email: Your registered email address
  • Plan Type: Current subscription plan
  • Signup Date: When you joined
  • Last Seen: Your last login timestamp
  • Credits: Available credits for your plan
  • Chatbots: Number of active chatbots
  • Personas: Number of personas (Gold/Platinum)

Last Seen Tracking

The platform automatically tracks your activity:

  • Updates when you login to the dashboard
  • Helps with account security monitoring
  • Used for usage analytics
  • Can help identify unauthorized access

üìà Activity Insights

Regular logins show active usage. If you notice login times you don't recognize, contact support immediately.

8. FAQ & Troubleshooting

Find answers to common questions and solutions to frequent issues.

Frequently Asked Questions

Q: I've reached my chatbot limit. What can I do?

Solutions:

  1. Delete existing chatbots: Remove chatbots you no longer need
  2. Upgrade your plan: Higher plans allow more chatbots
  3. Optimize current chatbots: Combine functionality into fewer bots
  4. Contact support: Discuss custom solutions for your needs
Q: I can't create personas. What's wrong?

Persona Requirements:

  • You must be on a Gold or Platinum plan
  • Free and Silver plans do not include persona features
  • Upgrade your plan to access persona creation
  • Check that you haven't reached your persona limit
Q: My chatbot isn't responding. What should I check?

Troubleshooting Steps:

  1. Check credits: Ensure your chatbot has available credits
  2. Verify API key: Make sure the API key is valid and active
  3. Check internet connection: Ensure stable connectivity
  4. Review error messages: Look for specific error information
  5. Test with simple queries: Try basic questions first
  6. Restart the chatbot: Refresh the page or clear cache
Q: How do I reset my password?
  1. Contact support via the provided email
  2. Provide your registered email for verification
  3. Follow the password reset instructions sent by support
  4. Create a new strong password
  5. Login with your new credentials
Q: Can I export my chatbot conversations?

Export capability depends on your plan:

  • Free: No export feature
  • Silver, Gold, Platinum: Yes, export is available
  • Look for export options in your chatbot interface
  • Export formats may include text, PDF, or other options

Session/Login Issues

"I can't login" or "Session expired" errors

Solutions:

  1. Clear browser cookies and cache
  2. Check your credentials: Verify email and password
  3. Account status: Ensure your account is active (not pending)
  4. Browser compatibility: Try a different browser
  5. Session timeout: Login again if session expired
  6. Contact support: If problems persist

Chatbot Performance Tips

Improving response quality
  • Use clear, specific questions
  • Provide context in your queries
  • Use appropriate AI models for your needs
  • Regularly update API keys
  • Monitor and manage credit levels
  • Clear conversation memory periodically
Speeding up responses
  • Use GPT-3.5 instead of GPT-4 for faster responses
  • Keep conversations focused and concise
  • Avoid overly complex queries
  • Ensure stable internet connection
  • Close unnecessary browser tabs

Credit Exhaustion Handling

"My chatbot ran out of credits"

Immediate Actions:

  1. Upgrade your plan: Get more credits instantly
  2. Reduce usage: Limit chatbot interactions
  3. Create new chatbot: Some plans reset credits for new bots
  4. Contact support: Discuss credit packages
  5. Optimize usage: Use credits more efficiently

Common Configuration Errors

Invalid API key errors
  1. Verify the API key is entered correctly
  2. Check that the API key is active and valid
  3. Ensure you have sufficient API quota
  4. Try regenerating the API key from your provider
  5. Update the key in your chatbot configuration
Color configuration not working
  1. Use valid hex color codes (e.g., #FF0000 for red)
  2. Ensure the # symbol is included
  3. Use 6-digit hex codes
  4. Clear browser cache after changes
  5. Save changes before testing

  • Include version numbers: "SupportBot_v2" for iterations
  • Use project codes: "ProjX_FAQ_Bot" for organization
  • Keep it readable: Avoid excessive abbreviations
  • Be consistent: Use similar patterns across related bots

Writing Effective Bot Personalities

  • Consistency matters: Maintain the same tone across conversations
  • Match your audience: Adapt personality to user expectations
  • Use personas: Create detailed character backgrounds
  • Test and refine: Adjust based on user feedback

Optimizing Credit Usage

  • Batch similar requests: Ask multiple related questions at once
  • Use concise language: Short, clear messages use fewer credits
  • Choose the right model: GPT-3.5 is cheaper and faster for simple tasks

Privacy and Security Best Practices
